## SYMPTOGRAM
-This MVP was made using Expo, React-Native and Firebase. 

## Installation

* **Clone and install packages**

```
git clone https://github.com/indritkello/Symptogram.git
cd Symptogram
npm install
npm install expo
npm install -g expo-cli
expo start
```
